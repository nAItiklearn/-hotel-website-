# Aurelius Riverside - Luxury Lakeside Hotel

A high-end, luxury boutique hotel website demonstration inspired by classic Italian lakeside elegance. This project showcases a sophisticated "magazine-style" editorial layout with a focus on immersive visual storytelling.

![Hero Preview](/assets/video/mainpage.mp4)

## âœ¨ Key Features

- **Cinematic Experience**: Autoplay, looping video background in the hero section for an immediate luxury feel.
- **Editorial Design**: Asymmetrical "collage" layouts with generous whitespace and overlapping elements, following modern luxury design principles.
- **Responsive Layout**: Seamlessly transitions from desktop to tablet and mobile, maintaining its premium aesthetic.
- **Micro-Interactions**: Smooth hover effects and transitions that enhance the user experience without being intrusive.
- **Luxury Typography**: A curated pairing of *Playfair Display* (Serif) for headings and *Inter* (Sans-serif) for body text.
